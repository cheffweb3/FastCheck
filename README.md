# FastCheck
FastCheck - Narrative fact-checker &amp; Intel Center for memecoin traders
Real-time fact-checking for tweets and memecoin narratives, powered by Grok AI and live web + X search.
Supported platforms:
X / Twitter
Axiom
Padre
GMGN

How it works:
The extension reads tweet text from the page and sends it to the FastCheck API, which uses Grok AI with live web and X search
to verify the claim and return a verdict. No data is stored by the extension — everything is handled server-side.

Getting started:
Install the extension from the Chrome Web Store
Open @FastCheckSolanaBot on Telegram and send /start
Get your auth token with /token
Paste the token into the extension to log in

Plans:
Basic - 5 Checks/Day
Basic - 100 Checks/Day
Pro - 300 Checks/Day

Note:
This repository contains only the Chrome extension frontend. The backend API and Telegram bot are proprietary and not included.
